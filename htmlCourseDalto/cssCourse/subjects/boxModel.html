Box Model


What's a box model?

The CSS box model is essentially a box that wraps around every HTML element. It consists of: content, padding, borders and margins.

Looks like this

4 main properties:

  1. Content

    2. The content CSS property replaces content with a generated value.


  2. Padding

    3. The padding CSS shorthand property sets the padding area on all four sides of an element at once.


  3. Border

    4. The border shorthand CSS property sets an element's border. It sets the values of border-width, border-style, and border-color.


  4. Margin

    5. The margin CSS shorthand property sets the margin area on all four sides of an element. Margin does not form part of the box, only separate the element between boxes.
